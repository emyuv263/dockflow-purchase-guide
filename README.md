## 这玩意儿到底能干嘛

DockFlow 的核心思路很直接：给 Dock 栏做"场景预设"。你想想，平时我们切换工作状态，不只是换个壁纸那么简单，而是整组应用都在变——从 Slack、Zoom 这种开会工具，切到 VS Code、终端这种干活工具，再切到 Spotify、Netflix 这种摸鱼工具。

DockFlow 让你把这些组合提前存好，起个名字，比如"早会模式""深度工作""周末躺平"，然后通过菜单栏图标、快捷键，甚至 Apple 快捷指令来一键切换。切过去的时候，它还能顺手把不在当前布局里的应用关掉，或者切换前自动保存当前布局，省得你来回折腾。

除了应用图标，你还能往 Dock 栏里塞文件夹、文件、网址链接，甚至加空白间隔把图标分组隔开，让整个布局看起来更整齐。配置也能导出导入，换电脑或者重装系统的时候，备份恢复一下就行。
